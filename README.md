# Dynamically Generated Typist Trainer
### Jeremy Thummel, Andrew Hunter, & Sam McAdoo

The goal of this project is to replicate and implement this [Paper](https://aclanthology.org/2023.acl-long.567/)

In essence, this paper contains two distinct algorithms: A SAKT (Self Attentive Knowledge Training) model, and a more typical language model which ingests the results found by the SAKT to produce new exercises.
